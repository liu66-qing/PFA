\begin{abstract}
3D medical Segment Anything Models promise interactive segmentation via point prompts, yet we discover they suffer from \emph{prompt unfaithfulness}: the model largely ignores prompt location and predicts based on image priors alone, achieving only 20.8\% Switch Accuracy---the rate at which predictions correctly change when prompts target different organs. This failure is distinct from prompt robustness (only 1.3-point Dice drop under 10-voxel noise) and is hidden by standard Dice evaluation. We identify the root cause as single-prompt training, which never requires the model to produce different outputs for different prompts on the same image. To address this, we propose \emph{Paired Prompt-Contrastive Training}: for each image, we sample two organ prompts and supervise both predictions through a shared LoRA-adapted decoder. Comprehensive mechanism ablation reveals that paired prompt exposure itself is the dominant factor (+40 points Switch Accuracy), while auxiliary losses (grounding regularization, switch margin) provide only marginal additional benefit (+0--5 points). Our method achieves Switch Accuracy 0.993$\pm$0.012 (from 0.208 baseline) with only 4.4-point Dice cost on AMOS across three seeds, generalizes to BTCV external validation (Switch 1.000), and adds merely 5.3\% training overhead with zero inference cost. Code will be released upon acceptance.
\end{abstract}
